同学们，我深知大家对于高GPA、高绩点的渴望，但请相信只要你交了作业，助教就会给你们满分的平时成绩，重要的不是你交上来了一份完美的作业，而是你真的学会了精算概论这门课，通过平时作业体现你的思考，期末考试不至于拿到题目连公式都记不起来。希望大家不要舍本逐末！
